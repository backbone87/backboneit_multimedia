=== Titel
Multimedia Integration


=== Suchbegriffe
multimedia,video,audio,barrierefrei,player


=== Teaser
Barrierefreie, Player-unabhängige Audio & Video Integration für Contao.


=== Beschreibung
<p><strong>Achtung: Sie müssen eine Player-Erweiterung für die Multimediatypen,
die Sie anzeigen möchten, installieren.</strong></p>

<h3>RC Phase</h3>
<p>Diese Erweiterung befindet sich in der RC-Phase und möglicherweise sind noch
nicht alle Features fehlerfrei nutzbar. Folgende Features brauchen noch
intensivere Tests:</p>
<ul>
<li>HTTP-Pseudostreaming Bitrateswitching</li>
<li>RTMP-Streaming Bitrateswitching</li>
</ul>

<h3>Eigenschaften:</h3>
<ul>
<li>
<li>Multimediatyp: Video (Unterstützung der Features ist Player-abhängig)
<ul>
<li>Mehrere Untertitel pro Video</li>
<li>Audio-Beschreibung</li>
<li>HTTP-Quellen (HTML5/Flash/Video-Download)</li>
<li>HTTP-Pseudostreaming-Quellen (Flash)</li>
<li>RTMP-Quellen (Flash)</li>
<li>SMIL-Quellen (playerabhängig)</li>
<li>Automatische Überprüfung der Quellen</li>
</ul>
</li>
<li>Multimediatyp: YouTube-Video (Unterstützung der Features ist Player-abhängig)
<ul>
<li>Akzeptiert alle offiziellen YouTube Video-Links
<li>Automatische Überprüfung der YouTube-Quelle</li>
<li>Titel, Beschreibung und Vorschaubild werden von YouTube abgefragt</li>
</ul>
</li>
<li>Automatische Ermittelung eines passenden Players für den jeweiligen Medientyp</li>
<li>Einfach Integration direkt auf Template Ebene.</li>
<li>Flexible API um eigene Medientypen und/oder Medienplayer zu integrieren.</li>
</ul>

<h3>Ausblick (Reihenfolge nicht signifikant)</h3>
<ul>
<li>Englische Übersetung</li>
<li>InsertTag Unterstützung</li>
<li>Integration in News, Kalender und als Modul</li>
<li>Mediabox Unterstützung</li>
<li>Multimediatyp: Audio</li>
<li>Multimediatyp: Video-Livestream</li>
<li>Video-Untertitel direkt in Contao pflegen</li>
</ul>
